# QuenchMap

QuenchMap е веб-апликација за прикажување, управување и категоризација на јавни чешми. Проектот користи HTML темплејти, CSS и JavaScript компоненти, PHP backend логика и поддржува повеќе јазици.

## Карактеристики
- ✔️ Динамички HTML/CSS/JS темплејти
- ✔️ PHP backend за обработка на податоци
- ✔️ Админ панел за управување со чешми и содржини
- ✔️ Подготвено за понатамошна интеграција со база на податоци

## Структура на проектот
- CSS, JS, слики
- HTML темплејти
- административен панел (во развој)
- PHP backend фајлови
